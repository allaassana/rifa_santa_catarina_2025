# Rifa Santa Catarina 2025
Plataforma de mobilização de recursos para o Centro Pastoral Santa Ana & São Joaquim.

## Ficheiros
- `index.html` — Página do participante/comprador
- `admin.html` — Painel do proprietário
- `style.css` — Estilos partilhados
- `script-participante.js` — Lógica da página do participante
- `script-admin.js` — Lógica da página do proprietário

## Uso
Abra `index.html` ou `admin.html` num navegador.
